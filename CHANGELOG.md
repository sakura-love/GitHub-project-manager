# Changelog

本文档记录 GitHub Project Manager 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2025-07

### 新增

- 本地 + 线上项目统一管理：同一界面管理本地 Git 仓库与 GitHub 在线仓库
- 自动加载当前账号拥有且非 fork 的仓库
- 项目内容可视化浏览：双击展开目录树，支持文件名搜索与内容预览
- Release 发布工作流：可视化填写 Tag、版本名、更新说明，支持上传多个资产文件
- Windows 11 风格界面：统一浅色主题，支持一键切换深色模式
- GitHub Token 使用系统 Keyring 保存，不写入仓库

### 后续计划

- 仓库克隆/拉取状态可视化
- Commit 历史与差异查看
- Issue / PR / Actions 能力扩展
- 自动更新与更完整打包流程
